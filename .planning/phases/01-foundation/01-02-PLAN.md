<feature>
  <name>Typed storage helpers (browser.storage.local)</name>
  <files>src/storage/storage.ts, tests/storage.test.ts</files>
  <behavior>
    saveApiKey(key):
    - Writes key to storage under STORAGE_KEYS.API_KEY
    - Overwrites any existing value

    getApiKey():
    - Returns the stored key string if present
    - Returns undefined if no key stored

    saveJd(jd):
    - Writes JD object under STORAGE_KEYS.jd(jd.id)
    - Appends jd.id to the JD_INDEX array if not already present
    - Does not duplicate the id if called twice with same jd.id
    - Preserves rawText field exactly as provided

    getAllJds():
    - Returns [] when no JDs saved
    - Returns all JDs in index order when JDs exist
    - Filters out any undefined/null values from corrupted storage

    deleteJd(jdId):
    - Removes the JD from storage (STORAGE_KEYS.jd(jdId))
    - Removes jdId from the JD_INDEX array
    - Clears ACTIVE_JD_ID if it equals jdId
    - Does NOT clear ACTIVE_JD_ID if a different JD was deleted

    setActiveJdId(jdId):
    - Writes jdId to STORAGE_KEYS.ACTIVE_JD_ID

    getActiveJdId():
    - Returns the stored jdId string if set
    - Returns undefined if not set

    Skill weight persistence:
    - A JD saved with skills containing weight: 'mandatory' retrieves with weight: 'mandatory'
    - A JD saved with skills containing weight: 'nice-to-have' retrieves with weight: 'nice-to-have'

    getStorageUsageBytes():
    - Returns a number (bytes in use from browser.storage.local.getBytesInUse)

    STORAGE_QUOTA_BYTES:
    - Exported constant equal to 10 * 1024 * 1024 (10MB)
  </behavior>
